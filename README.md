## R 언어의 특징
### - 데이터 분석에 특화된 언어
    - Python과 달리 데이터 분석에만 특화되있는 언어임
    - 컴파일 과정 없이 바로 실행하여 결과 확인 가능
    - R로 작성한 것은 프로그램이 아니라 스크립트라고 부름

### - 다양한 패키지(라이브러리) 제공
    - 패키지를 이용하여 시각화하여 데이터를 보여주기가 쉽다
    - 최신 이론이 발표되면 바로 R 패키지가 만들어지기 때문에 신속하게 최신 이론을 데이터 분석에 활용하는 것이 가능

### - 미적이고 기능적인 통계 그래프 제공
    - ggplot(아주 유명한 패키지)이라는 패키지를 통해 아름다우면서도 기능적인 통계 구현가능

### - 편리한 프로그래밍 환경
    - R 프로그래밍을 위한 통합 개발 환경으로 R 스튜디오가 제공되어 모든 작업을 R 스튜디오 내에서 처리 할 수 있음
    - 프로그램 작성, 실행, 수정 등 여러 작업을 수행하려면 보다 편리한 작업 환경이 필요함
    - 이러한 작업 환경을 통합 개발 환경, IDE(Integrated Develpment Environment)라고함
    - 코딩은 VScode로 하고 실행은 R에서 바로 실행 가능(교수님이 보통 이렇게 작업하심)
    - 맵이나 변수의 실시간 확인하기 위해 R 스튜디오는 필수적으로 사용해야함 (VScode에서 지원X, VScode는 코드의 가독성과 코딩의 편리성을 위해 사용)

### - 무료 사용
    - R은 무료로 사용 할 수 있는 소프트웨어

## R을 배우는 이유
### - 4차 산업혁명
    - 4차 산업혁명 : 인공지능, 빅데이터, 로봇, 사물인터넷 등 새로운 과학 기술이 정치, 경제, 문화 전반에 영향을 미치면서 이러한 변화를 수용하고 발전 가능성을 최대화시키는 시대로, 인공지능과 빅데이터가 핵심 기술로 인식됨
    - 4차 산업혁명의 중심은 데이터
    - 시대적 흐름은 데이터를 잘 다룰 줄 아는 기업과 개인이 경쟁에서 우위를 점할 수 있음을 의미함
    - 컴퓨팅 사고와 데이터 활용 능력은 전공을 불문하고 어떤 분야로 진출하든 점점 더 중요한 스펙임
    * 크고 많은 데이터도 빅데이터지만 데이터와 데이터 사이에서 벌어지는 일들 예를들어 어떤지역에서 사람들이 카드를 사용 -> 실제 데이터 : 사람이 남자인지 여자인지 나이는 몇살인지 어떤 음식을 먹었는지 어떤 음식점을 갔는지가 카드사 DB에 남음 그 정보와 기상청 데이터에서 날씨가 어땠는지 온도는 어땠는지등을 이용하면 새로운 데이터가 창출됨 그러한 데이터 가공 또한 빅데이터라고 함

    - 특별한 설정 없이 All Next 클릭하여 설치
    - 설치 다되면 시작에서 R 프로그램 클릭하여 UI 뜨는지 확인 (R과 R 스튜디오는 다름 R 프로그램 사용 안 할 거임)

### - R 스튜디오 설치
    - www.Rstudio.com 진입 후 Product 클릭 > R Studio IDE 클릭 > 우측 상단 R Studio Download 클릭 > RStudio Desktop의 Download R Studio 클릭 > Download R Studio Desktop for Windows 클릭
    - 디폴트 값으로 설치 진행

### - R 스튜디오의 화면 구성
    - 왼쪽 상단부터 소스영역, 환경영역, 콘솔영역, 파일영역으로 나뉨
    - 환경영역 : 변수값 확인가능 (자동출력됨)
    - 파일영역 : 출력이 나오는 영역
    - 콘솔영역 : 한줄씩 실행가능

### - 파일 영역
    - 해당 영역 File 탭에서 윈도우 탐색기와 동일한 역할 수행가능
    - 파일 영역에서 특정 폴더로 이동 후 More -> Set as working Directory 메뉴를 이용하여 작업 폴더로 지정가능
    - 플롯창 : R 명령문으로 작성한 그래프가 표시되는 영역
    - 패키지창 : R에서는 수많은 함수를 제공하고 있는데, 이러한 함수들은 작성 목적이나 개발자에 따라 패키지 형태로 제공

### - R 언어 예제
    - A <- 51:80
    - A라는 변수에 51부터 80까지 대입
    - 환경영역에서 확인가능하며 배열의 형태로 A [1:30] 으로 표시됨 배열 1번부터 30번까지 있다는 뜻
    - 소스영역에서 스크립트 작성 후 RUN 버튼 클릭시 커서가 위치한 곳부터 한줄씩 실행됨
    - 여러줄 한꺼번에 실행은 여러줄 선택 후 RUN 버튼 클릭하면 됨

### - 단축키
    - 한줄의 명령문 실행 : 커서를 특정 위치에 위치 시킨 후 Ctrl + Enter
    - 여러줄의 명령문 실행 : 여러줄을 선택 후 Ctrl + Enter

* 괄호 닫지 않으면 출력이 되지 않고 + 기호 나타남 (추가 입력 시까지 대기 -> 이때 Esc 키 입력 시 명령 입력 중단됨)
* R 언어는 대소문자 변수를 구분함
* 한 줄에 여러줄의 명령어 실행 시 ;으로 구분함
* 저장시 확장자 이름은 보통 .R로 저장
* R은 컴파일을 통해 .exe 형태의 실행 파일을 만들지 않고 명령문을 바로 실행하는 인터프리터 방식의 언어로, 이를 스크립트 언어라고 부름
* 주석은 #으로 표시함
* sqrt(25) -> 25의 제곱근
* max(A,B,C) -> A,B,C 중 최대값 구해줌 <->min
